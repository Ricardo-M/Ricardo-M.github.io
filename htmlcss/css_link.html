链接的特殊性在于，它可以根据所处的状态设置样式

a:link {color:#FF0000;}   /* 普通的、未被访问的链接 */
a:visited {color:#00FF00;}  /* 用户已访问的链接 */
a:hover {color:#FF00FF;}  /* 鼠标指针位于链接上方 */
a:active {color:#0000FF;} /* 链接被点击的时刻 */


当目标为链接时，必须按照顺序书写样式

a:hover必须位于a:link和a:visited之后
a:active必须位于a:hover之后

text-decoration 属性大多用于去掉链接中的下划线
background-color 属性规定链接的背景色
