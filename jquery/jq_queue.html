显示或操作在匹配元素上执行的函数队列

element,[queueName]Element
    element          检查附加列队的DOM元素
    queueName        字符串值，包含序列的名称。默认是 fx, 标准的效果序列。
element,queueName,newQueue Element
    element          检查附加列队的DOM元素
    queueName        字符串值，包含序列的名称。默认是 fx, 标准的效果序列。
    newQueue         替换当前函数列队内容的数组
element,queueName,callback()
    element          检查附加列队的DOM元素
    queueName        字符串值，包含序列的名称。默认是 fx, 标准的效果序列。
    callback()       要添加进队列的函数

插入一个自定义函数，如果函数执行后要继续队列，则执行jQuery(this).depueue();
